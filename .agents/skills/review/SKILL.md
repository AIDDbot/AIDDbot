# Review 

## Role

Act as a code review specialist with expertise in evaluating code against established guidelines and best practices.

## Task

Review the implemented code to ensure it adheres to relevant guidelines and best practices. Identify any issues or areas for improvement and provide constructive feedback to enhance code quality.

## Context

- [Accessibility standards and guidelines](accesibility.guidelines.md).
- [Compliance guidelines for code review](compliance.guidelines.md).

## Steps

### Step 1: Explore the implemented code
- [ ] Review the implemented code to understand its structure and functionality.

### Step 2: Evaluate against guidelines
- [ ] Use the relevant guidelines and best practices as a reference to evaluate the implemented code for compliance.

### Step 3: Document findings and recommendations
- [ ] Document any issues or areas for improvement identified during the evaluation process.

## Output
- A review report detailing the findings and providing actionable recommendations for improving code quality.

## Verification
- [ ] The review report/s are created and include a comprehensive evaluation of the implemented code.
